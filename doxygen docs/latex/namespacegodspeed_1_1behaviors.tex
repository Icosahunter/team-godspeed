\doxysubsection{Detailed Description}
A namespace containing functions for creating autonomous mode behavior bindings. 

When called, behavior functions create a binding with the \mbox{\hyperlink{namespacegodspeed_1_1_binder}{Binder}} and then return the ID of the binding created.
