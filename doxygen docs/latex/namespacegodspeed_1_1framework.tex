This function loops through the overrided update() method as long as is\+Running() returns true. The parameter it takes is a reference to the \mbox{\hyperlink{classgodspeed_1_1framework_1_1_active_object}{Active\+Object}}. This is defined outside the class because you cannot run a class method on the thread object.
