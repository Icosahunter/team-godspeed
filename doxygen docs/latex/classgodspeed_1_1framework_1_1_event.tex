\doxysubsection{Detailed Description}
A class for implementing observer patterns. 

The event class allows other objects to be notified when something happens. They do this by \char`\"{}subscribing\char`\"{} to the event by passing in a function that they wish to have called whenever that event happens. The event class will then go through it\textquotesingle{}s list of functions and call each one of them whenever it is raised.

It is the job of the object creating the event to raise the event when appropriate. Other objects that wish to subscribe to the event must necessarily have access to the event object or to a public function that can receive an event handler function.
