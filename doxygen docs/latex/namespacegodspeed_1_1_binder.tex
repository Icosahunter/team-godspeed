\doxysubsection{Detailed Description}
A namespace containing functions for binding value producing functions to value consuming functions. 

There are 3 types of functions considered here, source functions which take no values and return a double, pipe functions which take a double and return a double, and sink functions which take a double and return nothing. Once the \mbox{\hyperlink{namespacegodspeed_1_1_binder}{Binder}} is initialized, it will go through and repeatedly call the sink functions with the value produced by the source functions as the input.
