\doxysubsection{Detailed Description}
A namespace with a function for accessing the current number of balls stored in the robot. 

The \mbox{\hyperlink{namespacegodspeed_1_1inputs_1_1_ball_storage_a87c0ff594d45745536c26c8d44edd3a8}{Init()}} function must be called before the ball storage tracking can begin. That function attaches the inc and dec function as callbacks for when bumper A and bumper B are pressed (respectively).
