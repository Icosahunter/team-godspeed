\doxysubsection{Detailed Description}
A class for creating scripted paths for the robot to take. 

The paths are scripted using \char`\"{}commands\char`\"{} which are tuples consisting of an x-\/speed, a y-\/speed, an angular speed, and a duration. First create an instance of \mbox{\hyperlink{classgodspeed_1_1inputs_1_1_path_script}{Path\+Script}}, then add all commands you want, then call \mbox{\hyperlink{classgodspeed_1_1inputs_1_1_path_script_a36654438dc378f3d4da91520808c69f4}{Path\+Script\+::\+Execute\+Path()}} passing in the \mbox{\hyperlink{classgodspeed_1_1inputs_1_1_path_script}{Path\+Script}} you just created as the argument. This will immediately begin execution of the path. Path execution proceeds by recursive time-\/delayed updates which update variables, then whatever is bound the X(), Y(), and Angle() will have access to the updated variables. So the udpates are asyncronous from the actual movement changes and may not be deterministic.
